Projects

01

Business Data Miner

An easy-to-use terminal script that scrapes the entire United States directory of information on a particular industry. Retrieves business name, location information, phone number. This was used to map clustering businesses, saving researchers weeks worth of tedious Googling. Program was able to iterate and download 10,000 units of information in under 90 minutes through HTTP requests. I open sourced it on Github.

Technologies used: Beautiful Soup

Languages used: Python

Github Repo: https://github.com/HammadZ/pythonScraper

02

Crowd Music

Crowdmusic is a mobile and web application widget that syncs with the spotify to democratize music. Making use of a Geofence, the host can allow guests to vote on the next song to play. CrowdMusic allows party-goers to collectivley decide on the playlist. No more yelling across the room to the DJ or arguing with a group of friends for your favorite song. Submit a song and let the votes decide!

Technologies used: Bootstrap, AngularJS

Languages used: HTML, CSS, and JavaScript

Github Repo: https://github.com/HammadZ/

03

Worker Connect App Demo

App Demo for the Worker Connect Mobile App. The goal of the demo is to introduce the functionality of the app to the migrant workers. Video aimed to be translated into Hindi, Urdu, Arabic and others as part of the project. About the project: "In close collaboration with migrant workers, worker welfare staff, and support networks in the Persian Gulf, Worker Connect is a mechanism for migrant construction workers to learn about their rights, share information about their experiences, and provide anonymous feedback to worker welfare supervisors"

Technologies used: Final Cut Pro, Adobe Photoshop

04

StudyNow

StudyNow is an on-demand way to develop study groups based on proximity. It is both an academic enhancement application as well as a tool for social networking. Developed at HackathonEDU. My main role was UX/UI and functionality design, website development, mock-up iOS layout using graphic design tools and assess implementation.

Technologies used: Google Cloud platform, GoogleFirebase, web hosting and domain setup, The Foundation web framework, HTML, CSS, Javascript
