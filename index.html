Side Projects

Business Data Miner

An easy-to-use terminal script that scrapes the entire United States directory of information on a particular industry. Retrieves business name, location information, phone number. This was used to map clustering businesses, saving researchers weeks worth of tedious Googling. Program was able to iterate and download 10,000 units of information in under 90 minutes. I open sourced it on Github.

Technologies used: Beautiful Soup

Languages used: Python

Github Repo: https://github.com/HammadZ/pythonScraper


Crowdmusic

Crowdmusic is a mobile and web application that syncs with the spotify to democratize music. The host can allow guests to vote on the next song to play, keeping every event lit. Currently in Beta stages

Technologies used: Bootstrap, AngularJS

Languages used: HTML, CSS, and JavaScript
